关于《我虽然普通，但我妹妹是最强精灵》的书评

发表于 2023-11-16 分类于 生活 2k 2 分钟

作者辛苦了，我来锐评一下，有不同意见的尽管骂我：

一、感情线混乱

​ 除了主角和那对百合，对其他人的感情都描写的很牵强，陈景炎和南楠的感情非常的莫名其妙，在二人还在学院时，只有几笔带过，而等到南楠变成陈触不可及的人物之后，陈突然被作者赋予了对南楠的爱，可以说是完全不着边际。作为早期出现在主角团身边的一员，我甚至觉得全文只有寥寥几笔的慕容雪月对男主的感情都比光头的感情刻画的要真挚、丰富许多。

二、战斗场面混乱

​ 1、尤其是男主攻打魔法之源那一战，完全就是“剪不断、理还乱”，都要去救女主了，竟然只带了此时还没有成长起来的主角团，就像是为了给主角团“穿马甲”而故意让女主被抓一样，就是为了凸显主角团的重要性——画蛇添足。并且对战斗场景的描述，让读者看来总觉得头皮发麻，压根都不知道主角团的战力和反派的战力分别在什么地方，而且还拿出了一部分精力来描写魔都和帝国普通军队的“视死如归”，可以说是毫无必要。最后一战写了几十章，但是完全不知道在写什么。
​ 2、救出女主后并没有将女主进行转移或者派出主要战力来接应，在得知第一使魔已经回到蓝星的情况下还是只依靠主角团的薄弱力量在保护女主，并且紧随其后的描述中七大王权现在还都没有出手只是在天空城“待命”。一边说着“除了女主，这个世界它可以随时抛弃”，一边又用实际行动表达“我的天空城才是最重要的，女主，你谁啊？”完全不符合男主的设定（这里可能是作者自己的情感带入了）。
​ 3、最重要的救女主一战，乙方巅峰战力魔眼竟然没有出动。凭借魔眼的精神控制，完全可以做到不费一兵一卒拯救女主（只有一只五级变种人和一群科学家，而男主还可以困住五级变种人，不是睁睁眼的事情嘛）
​ 4、其实不难发现，描写战斗并不是作者的强项，可以看到文章前中期可以找到的关于战斗的场面描述可以说是屈指可数，如夏花般绚丽、短暂！作为以“爽”吸引读者的小说类型，作者在快要完结之时竟然想要凭借一己之力将小说强行拉到“传统玄幻”的小说类型中，天方夜谭嘛这不是，而且显而易见，作者并不擅长刻画“传统玄幻”中的战斗场面。打乱了爽文中仅存的逻辑并且使场面看起来非常的乱！！
​ 5、爽文就讲究一个爽就完了呗，甚至你说，最后男主三发歼星炮摧毁了一切然后和女主在太空要塞中过上了没羞没臊的生活（文中有提，虽然没建好，但已经可以使用）我都觉得很合理，写到最后，脱离了“爽文”的核心，却连“传统玄幻”的边都没摸到，可谓是“还没捡到芝麻，仅存的一个西瓜也丢了”

三、人物刻画单一

​ 或者说，没有人物刻画，主角的人物刻画也在一次次的冲突中被搅的残破不堪（干脆让主角也拥有双重人格得了）。所有人物的情感都几乎被作者一笔带过，一点细节都没有，然后在需要的时候直接被作者强行赋予，我记得比较清楚的一幕是——洛橙在天空之城哭泣，徐琴一脚踢爆自动贩卖机，两人拿着饮料坐在长椅，斜阳西下，拉长了两人的影子。明明是很温馨的画面却被作者写的如此生硬，就像是为了水文而特意诞生的毫无意义的一幕

四、文章篇幅短

​ 当然，以上的所有问题其实都是结果，小说烂的真正原因则是源自于——小说的篇幅太短了。
​ 短到作者没有时间去梳理全书的感情线，而是凭自己的想象随意捏造，需要了，就会随机赋予两个人感情，一点铺垫都没有，就像是爱神对于人类的肆意操纵，是强迫性的，身在书中的虚拟人物根本就无法反抗只能被迫接受；
​ 短到作者没有时间来进行战斗的描写，战斗是集中了“语言、动作、情感、心理、环境”描写的产物，是一个精华题，但是在本书中，似乎只有语言和动作以及关于战斗的省略（每次都是描写完战斗前的对峙，然后开始转换镜头，当镜头重新转移回来时，战斗已经结束了，关于战斗中的场景，哪怕是拥有上帝视角的读者也啥也不知道，只知道战斗胜利了）；
​ 短到作者没有时间去对人物进行详细的刻画，全书刻画的最多的人物男主，知道最后身为上帝视角的我们都不知道他的真实性格，永远猜不到他下一步要干什么，情感波动太大，甚至让我怀疑这本书是不是两个人写的，主角团那几个相对来说还算生动，但是其他人完全可以用“惨不忍睹”来形容。为了人物的细节而特意描写的人物细节，跟在大街上喊“我是马牛逼”有什么分别？
​ 关于两个人除了彼此对所有人都“没有情感”的早期设定，这本书完全可以用抛弃对其他次要人物的刻写，但是作者好像就是非要卖力的刻画次要人物，想要以500章的篇幅来完成对超宏大世界的叙事。要知道500章在传统玄幻里，第二个大世界都还没有出去吧。经典玄幻小说“完美世界”，花费了数章来对背景世界进行“白描”，而本书，竟然妄想直接将宇宙之上级别的宏大叙事几笔带过。……

五、总结

​ 总之，如此新颖的题材，如果本书能够理清人物关系，将人物刻画的再细致些，战斗场面描写的更详细、全面些，文章的篇幅再长些！我相信，这将是一部逼近“斗破苍穹”的国民级小说了。作为“爽文”，这本书除了结尾，其他都相当引人入胜（看不够，根本看不够）。但是总所周知，烂尾嘛，小说的通病了

RGB颜色

在 HTML 中，可以使用以下公式将颜色指定为 RGB 值：

rgb(red, green, blue)

每个参数（红色、绿色和蓝色）都定义为颜色的强度，其值介于0 - 255之间

于是一共有256 x 256 x 256=16777216种可能的颜色。

RGBA颜色

RGBA 颜色值是带有 Alpha 通道的 RGB 颜色值的扩展（增加了一个参数用来指定颜色的不透明度）

RGBA 颜色值指定为：

rgba(red, green, blue, alpha)

alpha 参数是介于0.0（完全透明）和1.0（完全不透明）之间的数字

在明年春天

发表于 2023-10-02 分类于 生活 1.5k 1 分钟

大雁大多飞走了，只留下黄昏的地平线在天际划出一句离别，渐隐在蝉的永夜。

我们之间的故事，像是一本读完的书，不会再有故事了。作为主角的你早已经落幕了，而我，不过是恋恋不舍的看着书的封底，迟迟没有翻过去的勇气······

我曾以为，我们的故事就像是阿尔丁尼与克里斯蒂娜，永远不会结束。然而，当我回首才发现，我只是一个家道中落的帕里斯，虽然我也深深的爱着朱丽叶，但在全书却也只享有无奈和绝望。你已经结束了你的篇章，而我只能站在原地，默默地回味着我们的过去。

我站在窗前，看着远方的天空，那里有一片孤独的云，像我们的故事一样，被留在寂寞的舞台上。我似乎听到了你的笑声，就像那飘渺的回声一样，逐渐消失在风中······

“啊多么痛的领悟~你曾是我的全部~只是我回首~来时路的每一步~都走的好孤独~”不知你是否发现，我的游戏昵称改为了“you were world”。是啊，你曾是我的全部，它代表着过去的美好回忆，也象征着我曾经拥有的孤独和失落。

我曾向朋友说说起你，我说“这就是无可救药的爱，互相的深情，只是我们都不擅长表达罢了”，然而朋友驳斥我“你只是因为太寂寞了”。可是我明明每天都跟你聊天啊，为什么会感到孤独呢？

我尝试去理解朋友的观点，我意识到，尽管我每天都在和你聊天，但这种交流仅仅是一种表面的互动，缺乏深层次的情感交流和理解。我开始思考我们之间的差异，为什么你总是能够保持冷静和理性，而我会轻易地被情绪所左右。我发现我们之间的交流更多地是一种信息的交换，而不是真正的情感表达。

——我将石头推向山顶，再任由其滚向山下，如此反复。我们一边与世界战斗，一边与世界和解，并逐渐为自身赋予明天的意义，“当荒谬的人深思他的痛苦时，他就使一切偶像哑然失声。荒谬的人知道，他是自己生活的主人”，或许，只有当我失去了在无尽徒劳中寻找意义的希望，这件事确是有了人生的意义······

扯远了，我终究不是一个满腹经纶的哲人，我的存在并没有意义，我从始至终就是一个无趣的人，我会为了两块面包而加班，我会为了一点尊严而争得头破血流。我没有目的，没有追求，更没有希望。我甚至没有梦想，因为我的梦，只会在深夜出现，只会在空荡荡的房间里，孤独地望着窗外，黯然神伤。我像一个机器人一样，按照预设的程序生活，没有任何情感，没有任何反应。我甚至不知道自己喜欢什么，我只知道我需要什么。我需要钱，我需要面包，我需要一切能让我活下去的东西。我不知道我是谁，我不知道我在哪里，我只知道我要努力工作，我要努力赚钱，我要努力生存。

我只是一个庸不可耐的俗人，误打误撞的跌入仙境，与你拥有了一段并不美好也不有趣的邂逅，天亮了，我就被驱逐了。你和我，就如同函数y=1/x的图像，分别在第一象限和第三象限游离。我们从无穷远的角落赶来，相互趋近，却始终未能交汇。短暂的驻足后，我们又各自朝着无穷远的方向离去，渐行渐远。可悲的是，我们既不能像平行线那般永远依偎，也不能如交叉线般短暂交汇。我们什么都做不到，我什么都做不到。恨人神之道殊兮，怨盛年之莫当，无奈，于是背下陵高，足往神留。遗情想像，顾望怀愁。

我们的故事就要结束了——在明年春天，在大雁返航的季节，我将合上这本属于我们的故事书。那场秋天的落叶，虽然美丽，但却无法再回到树上。我也只能默默地祝福你，希望你在未来的日子里能够开心快乐。我也将把那份爱，深深地埋藏在心里，成为我永远的记忆。

大雁大多飞走了，只留下寂寥的秋风在空中凌乱着书写着“祝你幸福”，沉寂在枫的怀抱。

希望你能看到吧
